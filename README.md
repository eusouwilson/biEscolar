# biEscolar
Analise de dados de uma escola e criação de um dashboard para acompanhamento das matrículas

Requisitos:

Quantidade de alunos matriculados<br>
 <li>  Por Nível </li> 
 <li>  Por Turma </li> 
 <li>  Por Séerie </li> 
 <li>  Tipo de Alunos(Rematriculas, Novos)</li> 
 <li>  Bolsistas(50% e 100%)</li>
